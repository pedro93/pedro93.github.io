Position: Summer Internship
Where: Critical Manufacturing
Role: Developing a STDF Comparator
Description: Created a unix diff-like utility tool and helped develop the underlying I/O (input and output) library to compare between 2 Standard Test Data Format (STDF) files.
Enables comparison between 2 15GB binary files in under 15 minutes with exhaustive search and comparison.
For reasonably small file sizes (~5-10MB) results are given in 2-3 seconds.
